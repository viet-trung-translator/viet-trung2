# along 翻译

Real-time **Chinese ↔ Vietnamese** speech translation web app. Two modes:

1. **Interpreted call** — two people in different places call each other; each hears the
   other in their own language.
2. **Single-device translate** — two people sharing one phone, taking turns.

Runs in any mobile/desktop browser (no install). Built per [`GIOI-THIEU-APP.md`](GIOI-THIEU-APP.md).

## Stack

- **Frontend:** React + Vite + TypeScript, Web Audio API (PCM 16k in / 24k out).
- **Backend:** Node.js + Fastify + WebSocket (`@fastify/websocket`).
- **DB:** PostgreSQL (Neon) — accounts + call history.
- **Auth:** JWT + bcrypt; first account becomes owner; others need owner approval.
- **AI:** Google Gemini Live (speech-to-speech) via `BidiGenerateContent` WebSocket.
- **Deploy:** Docker on Render (single service serves the SPA + API + WS).

## Project layout

```
  auth/                register/login/JWT + admin routes
  contacts/            frequent contacts + search routes

  src/realtime*.ts(x)  WS client + call/solo state

```

## Local development

Prerequisites: Node 20+, a PostgreSQL database (Neon works), and a Gemini API key
(optional — without it the app runs in **echo mode** so you can test the audio
pipeline end-to-end without translation).

```bash
cp .env.example .env        # fill in DATABASE_URL, JWT_SECRET, GEMINI_API_KEY
npm install
npm run migrate             # create tables (or it runs automatically on boot)
npm run dev                 # server :8080 + client :5173 (Vite proxies /api and /ws)
```

Open http://localhost:5173. The first account you register becomes the **owner**
and is auto-approved; subsequent accounts wait in the owner's **Admin** page.

> **Note on `GEMINI_MODEL`:** the spec targets `gemini-3.5-live-translate-preview`.
> If that model id is not available on your key, set `GEMINI_MODEL` to a live model
> you have access to (the integration uses the standard Gemini Live audio protocol).

## Production build

```bash
npm run build      # builds client/dist and dist/server
npm start          # node dist/server/index.js (serves SPA + API on $PORT)
```

## Deploy to Render

This repo includes a `Dockerfile` and `render.yaml`. Create a new **Web Service**
from the repo (Docker runtime) and set the secret env vars:

- `DATABASE_URL` — Neon connection string (`?sslmode=require`)
- `GEMINI_API_KEY` — Google Gemini API key
- `GEMINI_MODEL` — live translate model id
- `JWT_SECRET` — auto-generated by `render.yaml`

The free plan sleeps after ~15 min idle (first request then takes 30–60s).

## Usage notes

- **Take turns when on speakerphone** (half-duplex): the mic is muted while the
  translation is playing to avoid echo/feedback. Use earphones to talk continuously.
- Both users must have the app open to receive a call (no push notifications yet).
